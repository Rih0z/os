# 学習ノート

このフォルダには、各フェーズで学んだことを記録します。

## フォルダ構成

```
notes/
├── phase1/   - ブートプロセス
├── phase2/   - プロセス管理
├── phase3/   - メモリ管理
├── phase4/   - プロセス間通信（IPC）
├── phase5/   - ファイルシステム
├── phase6/   - デバイスドライバ
└── phase7/   - Linuxカーネル（発展）
```

## ノートの書き方

各ノートは以下のテンプレートを参考に書いてください：

```markdown
# トピック名

## 学習日
YYYY-MM-DD

## MikanOSでの実装
（MikanOSでどう実装されていたかを振り返る）

## MINIX 3での実装
（MINIX 3のソースコードを読んで理解したこと）

## 比較・考察
（2つのアプローチの違いと、なぜそうなっているかの考察）

## 疑問点
（まだわからないこと、次回AIに聞くこと）

## 参考
（参照したソースコードのパス、書籍のページ番号など）
```
